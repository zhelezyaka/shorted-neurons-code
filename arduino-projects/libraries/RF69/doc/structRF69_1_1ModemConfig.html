Detailed Description

Defines register values for a set of modem configuration registers.

Defines register values for a set of modem configuration registers that can be passed to setModemConfig() if none of the choices in ModemConfigChoice suit your need setModemConfig() writes the register values to the appropriate RF22 registers to set the desired modulation type, data rate and deviation/bandwidth.
